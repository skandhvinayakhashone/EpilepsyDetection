# EpilepsyDetectionSystem
Epilepsy Detection System with EEG data using LSTM and XGBoost

Data Set : https://www.isip.piconepress.com/projects/tuh_eeg/downloads/tuh_eeg_epilepsy/

Requirements:

keras >= 2.3.0

numpy >= 1.18.0

matplotlib >= 2.0.0

mne >= 0.18.0

pandas >= 0.23.0

scikit-learn >= 0.20.0

Tensorflow >= 1.15.0

xgboost >= 0.7.0
